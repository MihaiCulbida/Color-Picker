# Color-Picker
A simple browser tool that lets you pick colors and copy their hex code.

#Pick a color from the wheel
1) Click the color input at the bottom of the widget
2) Choose any color
3) Click the hex box to copy it

#Pick a color from an image
1) Click Select Images to upload a photo, or paste one with Ctrl+V
2) Click the eyedropper button that appears
3) Hover over the image and click on the color you want
